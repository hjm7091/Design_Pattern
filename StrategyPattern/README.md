### 스트래티지 패턴

* 스트래티지 패턴은 전략을 쉽게 바꿀 수 있도록 해주는 디자인 패턴이다.
* 스트래티지 패턴은 같은 문제를 해결하는 여러 알고리즘(방식)이 클래스별로 캡슐화되어 있고 이들이 필요할 때 교체될 수 있도록 함으로써 동일한 
  문제를 다른 알고리즘으로 해결할 수 있게 하는 디자인 패턴이다.
